TGA Image Compression

A comprehensive implementation of TGA (Targa) image compression using Run-Length Encoding (RLE). This project was developed as part of my university coursework (UPR - Introduction to Programming).

Key Features:
  • RLE compression algorithm for efficient image storage
  • Support for uncompressed TGA format reading and writing
  • Command-line interface for easy file processing
What I Learned:

This project deepened my understanding of binary file formats, compression algorithms, and low-level data manipulation in C.
